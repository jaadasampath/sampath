2.Project Title :
Leveraging a hybrid deep learning framework for lung cancer detection

Description

This project focus on using deep learning frameworks to detect lung cancer from CT scans . It aims to help doctors to find cancer quickly and accurately. This project uses CNN(convolutional neural netwok ) to find patterns and Bi-directional LSTM (long short term memory) for understand changes over time.

Technologies

  • Python is the main programming language used in this program.
  • Using Python with TensorFlow to build CNN and LSTM models. CNN is used to analyze CT scan images
  • Using Python modules like SimpleITK or OpenCV to load and process the CT scan images into a suitable format for analysis.
  • Python uses NumPy and pandas for resizing images and labeling tumor areas.
  • Visualizing the data using matplotlib through graphs or loss curves.
  • Python scripts are used to train the CNN and LSTM models.
  • VS Code is the development environment.
